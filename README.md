# 🛡 Safety Bot — Бот безпеки

Бот для перевірки що людина в порядку. Кожну годину надсилає кнопку "✅ Я в порядку". Якщо не відповіла за 10 хвилин — надсилає тривогу на екстрені контакти.

## Деплой на Railway

1. Створи бота у @BotFather, отримай токен
2. Завантаж файли на GitHub (bot.py, requirements.txt, Procfile)
3. Зайди на railway.app → New Project → Deploy from GitHub
4. Додай змінну середовища: `BOT_TOKEN = твій_токен`
5. Railway сам запустить бота

## Як користуватись

### Захищена особа (подруга):
1. Знайти бота в Telegram, написати `/start`
2. Додати екстрені контакти: `/add_contact 123456789`
3. Запустити моніторинг: `/monitor_on`

### Екстрені контакти:
1. Теж мають написати `/start` боту (інакше не зможе надіслати повідомлення)
2. Отримають автоматичне повідомлення коли їх додадуть

## Команди

| Команда | Опис |
|---------|------|
| `/start` | Запустити та зареєструватись |
| `/monitor_on` | Увімкнути моніторинг |
| `/monitor_off` | Вимкнути моніторинг |
| `/add_contact ID` | Додати екстрений контакт |
| `/remove_contact ID` | Видалити контакт |
| `/contacts` | Список контактів |
| `/interval 30` | Змінити інтервал (хвилини) |
| `/status` | Статус бота |
| `/check_now` | Надіслати перевірку зараз |

## Дізнатись Telegram ID

Переслати будь-яке повідомлення боту @userinfobot — він покаже ID.

## Налаштування за замовчуванням
- Інтервал перевірок: 60 хвилин
- Час на відповідь: 10 хвилин
